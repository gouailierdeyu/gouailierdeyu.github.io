ADMM交替方向乘子法

发表于 2020-03-20 分类于 优化问题 阅读次数：

简单无约束凸优化问题：

minx  f(x).min_x\ \ f(x). minx​  f(x).

xxx是优化自变量，通过xxx变化，使得目标函数f(x)f(x)f(x)的数值达到最小。
对xxx没有任何取值约束，无约束优化。同时我们只考虑凸函数优化，凸函数才最容易找到最优解。

实际上我们对xxx的取值有很多要求，要满足一定的条件。
等式约束：

subject.to   Ax=bsubject.to \ \ \ Ax=b subject.to   Ax=b

不等式约束：

subject.to   Ax≤bsubject.to \ \ \ Ax≤b subject.to   Ax≤b

ADMM交替方向乘子法解决的优化问题：

minx,z  f(x)+g(z)sub.to  Ax+Bz=cmin_{x,z} \ \ f(x)+g(z) \\sub.to \ \ Ax+Bz=c minx,z​  f(x)+g(z)sub.to  Ax+Bz=c

ADMM通常解决等式优化，同时优化问题还有zzz和xxx两个优化变量。

分布式中的一致性优化问题（consensus），分享问题（sharing problem）可以用如上数学形式化，因为每个节点的变量还要跟周围节点变量产生关联，所以用ADMM的原因，是可以将部署同一个节点的变量分开计算，ADMM解决优化问题特别快。
使用增广拉格朗日乘子法，将约束条件合入目标函数变成：

Lρ(x,z,y)=f(x)+g(z)+yT(Ax+Bz−c)+(ρ/2)∣∣Ax+Bz−c∣∣22L_\rho(x,z,y)=f(x)+g(z)+y^T(Ax+Bz-c)+(\rho/2)||Ax+Bz-c||_2^2 Lρ​(x,z,y)=f(x)+g(z)+yT(Ax+Bz−c)+(ρ/2)∣∣Ax+Bz−c∣∣22​

将约束变量变成了x,y,zx,y,zx,y,z三个变量了。
(ρ/2)∣∣Ax+Bz−c∣∣22(\rho/2)||Ax+Bz-c||_2^2(ρ/2)∣∣Ax+Bz−c∣∣22​表示二次二范数惩罚项，加入惩罚项使函数更凸。
此优化问题通常使用迭代法求解：

step1:  xk+1:=arg minxLρ(x,zk,yk)step2:  zk+1:=arg minzLρ(xk+1,z,yk)step3:  yk+1:=yk+ρ(Axk+1+Bzk+1−c)step1:\ \ x^{k+1}:=arg\ min_xL_\rho(x,z^k,y^k)\\ step2:\ \ z^{k+1}:=arg\ min_zL_\rho(x^{k+1},z,y^k)\\ step3:\ \ y^{k+1}:=y^k+\rho(Ax^{k+1}+Bz^{k+1}-c)step1:  xk+1:=arg minx​Lρ​(x,zk,yk)step2:  zk+1:=arg minz​Lρ​(xk+1,z,yk)step3:  yk+1:=yk+ρ(Axk+1+Bzk+1−c)

一般情况下可以化简上述式子，可以将∣∣Ax+Bz−c∣∣22||Ax+Bz-c||_2^2∣∣Ax+Bz−c∣∣22​配方。

ADMM的迭代思想就是分别固定另外两个变量，更新其中一个变量：(也就是其名：交替方向)。

解法就是在目标函数中将xxx，zzz分别考虑，在xk+1x^{k+1}xk+1迭代时，将zzz看成是定量，对arg minxLρ(x,zk,yk)arg\ min_xL_\rho(x,z^k,y^k)arg minx​Lρ​(x,zk,yk)的计算，zzz是不影响求得使函数值最小得时候的xxx值。同理，在zk+1z^{k+1}zk+1迭代时，将xk+1x^{k+1}xk+1视为定量，不影响求使得函数最小时的zzz。初始化可以从0开始。
